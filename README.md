# Signal Decoder Technical Assessment

## Overview

Build an ASP.NET Core 8 Web API that decodes combined signals to identify transmitting devices.

## Requirements

- .NET 8 SDK
- Three projects: Domain, Application, API
- See [SPECIFICATION.md](docs/SPECIFICATION.md) for full requirements
- See [API_CONTRACT.md](docs/API_CONTRACT.md) for API endpoints

## Submission

1. Fork this repository
2. Implement your solution
3. Push to trigger automated tests
4. Check Actions tab for results

## Performance

- 10 devices: < 1s
- 15 devices: < 3s
- 20 devices: < 5s

## Evaluation

Tests validate correctness, performance, tolerance handling, and API compliance.
